Built on AWS

Serverless by design

Zero servers to manage. Pay only for what you use. Infinitely scalable.

⚡

API Gateway

REST endpoint that receives alert submissions, validates auth via API key, and routes to Lambda.

λ

AWS Lambda

Two functions: SaveAlert (on submit) and CheckPrice (on cron). Python 3.12, cold start under 200ms.

🗄️

DynamoDB

Stores alerts and price history with TTL-based expiry. Single-table design with composite keys.

⏰

EventBridge

Cron rule cron(0 */6 * * ? *) fires price checks every 6 hours automatically.

📣

SNS

Publishes alert messages when a price drop is detected. Fanout to multiple subscribers.

📧

Amazon SES

Sends formatted HTML email alerts to users. Domain verified, SPF/DKIM configured.
